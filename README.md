# configs
Some configurations I use with sane

## Repository index

```bash
sane apply Azer0s/configs/index
```

## Kafka

```bash
sane start kafka
```

## Couchbase

```bash
sane start couchbase
```

## ELK Stack

```bash
sane start elk
```

## Nginx

```bash
sane start nginx
```

## VSCode settings

```bash
sane apply vscode-settings
```
